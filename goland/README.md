# Instructions

## Run and Build

If only want to run the file
```shell
# run with
go run name_file.go
```

For building and running
```shell
# build with
go build name_file.go

#  name_file will be generated
./name_file
```

Example:

```shell
go run seo-scraper.go https://salrion.vercel.app
```
